üöÄ How to Use This API

Step 1: Upload DOCX Files

POST/api/v1/jobs

Upload one or more DOCX files to start the conversion process.

cURL Example:

curl -X POST "http://localhost:8000/api/v1/jobs" \ -H "accept: application/json" \ -H "Content-Type: multipart/form-data" \ -F "files=@document1.docx" \ -F "files=@document2.docx"
Response: You'll get a job ID to track the conversion progress.

Step 2: Check Job Status

GET/api/v1/jobs/{job_id}

Monitor the conversion progress and see when files are ready.

cURL Example:

curl "http://localhost:8000/api/v1/jobs/YOUR_JOB_ID"

Step 3: List Available Files

GET/api/v1/jobs/{job_id}/files

Get a list of all files with their download links.

cURL Example:

curl "http://localhost:8000/api/v1/jobs/YOUR_JOB_ID/files"

Step 4: Download Individual PDFs

GET/api/v1/jobs/{job_id}/files/{filename}/download

Download specific PDF files directly (not as ZIP).

cURL Example:

curl "http://localhost:8000/api/v1/jobs/YOUR_JOB_ID/files/document1.docx/download" \ -o "document1.pdf"

Step 5: Download All Files as ZIP (Alternative)

GET/api/v1/jobs/{job_id}/download

Download all converted files as a ZIP archive.

cURL Example:

curl "http://localhost:8000/api/v1/jobs/YOUR_JOB_ID/download" \ -o "converted_files.zip"
